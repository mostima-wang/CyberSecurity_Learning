# 🛡️ CyberSecurity Learning | 网安学习笔记

欢迎来到我的网络安全学习仓库！本项目用于记录我在渗透测试学习过程中的理论知识沉淀与实战案例分析。

## 📂 目录结构

### 📘 [渗透理论知识学习](./渗透理论知识学习/)
本模块涵盖了 Web 攻防的核心原理与常见漏洞分析：
* **[01-信息收集](./渗透理论知识学习/01-信息收集.md)**：包含搜索引擎利用、域名资产发现、CDN 绕过及端口探测技巧。
* **[02-Web攻防-文件类](./渗透理论知识学习/02-Web攻防-文件类.md)**：深度解析文件上传绕过（00截断、htaccess）、文件下载及包含漏洞（PHP伪协议利用）。
* **[03-Web攻防-注入类](./渗透理论知识学习/03-Web攻防-注入类.md)**：涵盖 SQL 注入分类（联合查询、报错、盲注）及 Sqlmap 实战用法。
* **[04-Web攻防-命令与脚本](./渗透理论知识学习/04-Web攻防-命令与脚本.md)**：记录 RCE 命令执行漏洞原理、XSS 跨站脚本、CSRF 伪造等漏洞。
* **[05-Web攻防-其它与逻辑](./渗透理论知识学习/05-Web攻防-其它与逻辑.md)**：包含 XXE 注入、SSRF 服务端请求伪造及高频业务逻辑漏洞（支付、越权、密码重置）。

### 🏗️ Tryhackme实战学习笔记
* **[01-Linux基础](./thm实战学习/01-Linux基础.md)**：包含Linux基础知识及基础命令。
* **[02-Windows&AD基础](./thm实战学习/01-Windows基础.md)**：包含Windows与active directory的基础知识及基础命令。
* **[03-命令行](./thm实战学习/03-命令行.md)**：包含Windows CMD、PowerShell、Linux Shell的基础命令。
* **[04-网络](./thm实战学习/04-网络.md)**：包含网络的基本知识，例如OSI，TCP/IP，各种协议等等。
* **[05-密码学基础](./thm实战学习/05-密码学基础.md)**：包含古典密码、公钥密码、数字证书等等。
* **[06-漏洞利用基础](./thm实战学习/06-漏洞利用基础)**：包含多个漏洞的相关知识、利用方法、metasploit的基础使用，以及一次简单的全流程渗透wp。
* **[07-Web应用程序基础知识](./thm实战学习/07-Web应用程序基础知识.md)**：包含 Web 应用程序基础（前后端组成）、URL 结构深度解析、HTTP 协议详解（消息格式、请求方法、状态码及安全首部）等核心知识。
* **[08-JS-SQL-BP](./thm实战学习/08-JS-SQL-BP.md)**：包含 JavaScript 交互与安全开发实践、SQL 数据库操作与常用查询模板、以及 Burp Suite 核心模块（Proxy、Target、Repeater 等）的使用指南。
* **[09-攻击性安全工具](./thm实战学习/09-攻击性安全工具.md)**：精炼了 Hydra 暴力破解、Gobuster 目录与子域名枚举的实战指令；系统梳理了正/反向 Shell 的原理、监听器配置及多语言 Payload；并总结了 SQLMap 自动化注入与数据提取的常用工作流。
* **[10-防御安全](./thm实战学习/10-防御安全.md)**：涵盖 SOC 运维与 5W 告警分选流程；详解 SANS/NIST 事件响应阶段与磁盘/内存取证工具（如 Volatility）；总结 Windows 核心事件 ID 与 Linux 命令行日志审计实战。
* **[11-安全解决方案](./thm实战学习/11-安全解决方案.md)**：解析 SIEM 日志集成与关联分析原理；解析防火墙的工作机制，解析 IDS 的监测阻断机制；汇总 Nessus 等漏洞评估工具及 CVE/CVSS 风险评分标准。
